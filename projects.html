FreeRTOS Timeline Scheduler

Extended the FreeRTOS kernel with a custom cyclic executive scheduler for ARM Cortex-M3 architectures. Engineered a strict lifecycle manager and an optimized trace system to guarantee deterministic execution for both hard and soft real-time tasks, validated via QEMU.

Secure Messaging Application

Self-Hosted Cloud Storage

AI Tax System Assistant
